Description

Widgets like @(files), @(pwd). To create a widget:

stick the code below implementing ParserWidget; create a prototype in Widget.h; in Parser.c, add to the symbol table, sym[] with the symbol you want, in ASCIIbetical order.

See Parser for more information.
